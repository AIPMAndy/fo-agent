## 💡 What is fo-agent?

A **browser-based AI Agent visual workspace** that lets you build, debug, and deploy AI personas locally — in minutes.

- **Zero barrier**: No coding required, **open browser and go**
- **Multi-model support**: OpenAI / Claude / Gemini / Ollama
- **Dify sync**: Push agents to Dify as a backend service with one click
- **Interactive Playground**: WYSIWYG AI chat debugging interface

## ⚡ Quick Start

### Option 1: Try the Demo (No Install)

👉 **[Open fo-agent Demo](https://aipmandy.github.io/fo-agent/)**

> Perfect for quick testing — no API Key needed

### Option 2: Run Locally

```bash
# Clone the repo
git clone https://github.com/AIPMAndy/fo-agent.git
cd fo-agent

# Open index.html in your browser
open index.html
```

### Connect Your AI Model

1. Open the fo-agent interface
2. Go to **Settings**
3. Select your AI provider (OpenAI / Claude / Gemini / Ollama)
4. Enter your API Key
5. Start building!

> ⚠️ **Privacy Notice**: Your API Key stays in your browser only — **never uploaded to any server**.
